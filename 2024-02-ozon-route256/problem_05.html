Условие задачи

Ваня очень любит играть в компьютерные игры, но у него нет денег, чтобы купить игру KTA5. Он решил скачать её с интернета, но вместе с игрой случайно скачал и вирус. Помогите Ване определить количество зараженных файлов, которые нужно удалить. Файлы и папки считаются зараженными, если они находятся в той же директории, что и вирус. Если папка заражена, это означает, что все файлы и папки внутри тоже заражены. Вирусный файл считается зараженным файлом. Все вирусные файлы имеют расширение .hack\texttt{.hack}.hack. Например: cat.png.hack\texttt{cat.png.hack}cat.png.hack, a.cpp.hack\texttt{a.cpp.hack}a.cpp.hack считаются вирусом, config.yaml\texttt{config.yaml}config.yaml, hack.cpp\texttt{hack.cpp}hack.cpp не считаются вирусом.

Входные данные

Каждый тест состоит из нескольких наборов входных данных. Первая строка содержит целое число ttt (1≤t≤1001 \le t \le 1001≤t≤100) — количество наборов входных данных. Далее следует описание наборов входных данных. Первая строка каждого набора входных данных содержит целое число nnn (1≤n≤20001 \le n \le 20001≤n≤2000) — количество строк с описанием JSON-а директорий. Следующие nnn строк каждого набора входных данных содержат описание JSON-а директорий. Все имена файлов и папок непустые, состоят из строчных латинских букв и точек (знаков ‘.’\lq\texttt{.}\text{\textquoteright}‘.’ ) и имеют длину не больше 202020. В описании могут быть символы пробела и табуляции. Гарантируется, что каждый набор входных данных содержит не более 400400400 папок и не более 101010 файлов в каждой папке. Гарантируется, что размер входных данных не превосходит 10Мб.

Выходные данные

Выведите ответ — число, количество зараженных файлов, которые нужно удалить. Вирусный файл считается как зараженный файл. Ответ для первого теста равен 3: kta.exe\texttt{kta.exe}kta.exe, kta.hack\texttt{kta.hack}kta.hack и cat.png.hack\texttt{cat.png.hack}cat.png.hack являются зараженными файлами. Любые внешние библиотеки использовать нельзя. ∙\bullet∙ Для работы с JSON в языке Go можно пользоваться библиотекой "encoding/json": https://pkg.go.dev/encoding/json\url{https://pkg.go.dev/encoding/json}https://pkg.go.dev/encoding/json.
